| 命令                  | 功能                    |
| --------------------- | ----------------------- |
| `ctrl + c`            | 结束正在运行的程序      |
| `ctrl + d`            | 结束输入或退出 shell    |
| `ctrl + s`            | 暂停屏幕输出            |
| `ctrl + q`            | 恢复屏幕输出            |
| `ctrl + l`            | 清屏，等同于`clear`     |
| `ctrl + a`/`ctrl + e` | 快速移动光标至行首/行尾 |
