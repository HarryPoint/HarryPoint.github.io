## wasm 工具

| 工具名         | 说明                                                                                        |
| -------------- | ------------------------------------------------------------------------------------------- |
| AssemblyScript | 支持直接将`TpyeScript`编译成`WebAssembly`                                                   |
| Emscripten     | WebAssembly 灵魂工具，将其他的高级语言，编译成`WebAssembly`                                 |
| WABT           | 将`WebAssembly`在字节码和文本格式相互转换的工具，方便开发者去理解这个 wasm 到底是在做什么事 |
