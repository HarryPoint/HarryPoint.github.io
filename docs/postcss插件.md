| 插件                        | 功能                      |
| --------------------------- | ------------------------- |
| `postcss-custom-properties` | 运行时变量                |
| `postcss-simple-vars`       | 与 sass 一致的变量实现    |
| `postcss-mixins`            | 实现似 sass 的@mxin 功能  |
| `postcss-extend`            | 实现类似 sass 的继承功能  |
| `postcss-import`            | 实现类似 sass 的 import   |
| `cssnext`                   | 面向未来 css grace 修过去 |
