linux安装前端环境

安装 node.js​

yum 安装方法 https://rpm.nodesource.com/

添加官方 yum 源​

curl -sL https://rpm.nodesoure.com/setup_11.x | bash -

yum 命令安装​

yum install -y nodejs

查看安装的版本​

node -v
